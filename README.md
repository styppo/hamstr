# Hamstr

[Hamstr](https://hamstr.to) is a twitter-style [Nostr](https://github.com/fiatjaf/nostr) web client.

### Install the dependencies
```bash
yarn
# or
npm install
```

### Start the app in development mode
```bash
npm run dev
```


### Lint the files
```bash
yarn lint
# or
npm run lint
```


### Format the files
```bash
yarn format
# or
npm run format
```


### Build the app for production
```bash
npm run build
```
